attrubute:

  • type:define the input format.
    • text: a textbox for inputting the text data.
    • password:also a textbox for text input, but it will hide the input text.
    • radio:a circle icon for clicking. if there are multiple radio icon, it can only click one.
    • checkbox:a square icon for clicking, it can be clicked for multiple option.
    • file:a input format for uploading the file from local to internet.
    • button:a button for click, it can be used on many usages.
    • submit:a button, activate it will send the whole data in the form to the action web or application.
    • reset:a button, activate it will reset all input to the default.
  • value:
    • when using on text, password:the default input.
    • when using on option, checkbox:the value of the choices.
    • when using on button-like object:the value showing on a button.
  • placeholder:a default or an example of input, different from the value in text, it won't be a real input.
  • checked: set a redio or checkbox to activated.
  • size:a lenght of a textbox.
  • mexlength:the maximum input of a textbox.
  • disabled:set a input being disabled for user.

select:

a drop-down menu with defined option.

  • option:the choice in a select tag.
  • multiple:specifies that multiple choice can be choosen at once.
  • selected:set an option being choosen.

textarea:

a block for paragraph input.

  • width:the width of textarea.
  • height: the height of textarea.
