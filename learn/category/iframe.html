iframe

define an embedding window in a web page.


demonstrate:

  • width:the width of embedding window.
  • height:the height of embedding window.
  • src: the resource of embedding window.
  • frameborder:specifies whether window has a border or not.("0"->off "1"->on)
  • allow: imformation about policy and privacy.
  • allowfullscreen: allow to look embedding video or web in fullscreen.
