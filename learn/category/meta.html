meta

the tag is about file imformation in head block.

  • charset:setting the unicode the file use.(for example:UTF-8, Big-5.etc)
  • name:writing the imformation for file.
    • application-name: the name of this file.
    • description:the intruction of this file.
    • keyword:the keyword when searching in the internet.
    • author:the writer of this file.
    • generator:the editor that this file use.
    • viewpoint:a cross platform window setting,for fitting the screen of different devices.
      • width:the width of website.
      • inital-scale:the inital zoom level of website.
  • http-equiv:the internet setting of the file.
    • refresh:this instruction will automatically reload the page.
  • content:the vaule of the last attribute,for example:
    • refresh:the content will be setting the time for refresh.
    • viewpoint:setting screen state when surfing the web page in differet platform device.
    • name:describe the last attribute.(except viewpoint)
  • URL:address of destination.(for example: web, file,etc)
